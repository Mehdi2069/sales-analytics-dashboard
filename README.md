# Sales Analytics Pipeline & BI Dashboard

An end-to-end sales analytics project built with Python.

## Features

- Synthetic sales data generation
- Data quality audit
- Data cleaning pipeline
- KPI analysis
- Data visualization
- Automated reporting
- Interactive dashboard with Streamlit

## Tech Stack

- Python
- Pandas
- NumPy
- Matplotlib
- Streamlit
- Faker

## Project Structure

sales_analytics_project/
├── data/
├── scripts/
├── output/
├── dashboard.py
├── main.py

## Workflow

Generate Data → Audit → Clean → Analyze → Visualize → Report → Dashboard

## How to Run

### Generate dataset
python scripts/generate_data.py

### Run analytics pipeline
python main.py

### Launch dashboard
streamlit run dashboard.py
